# Security Sessions

Notes, structure, and supporting material from live sessions and talks I've given on cybersecurity topics — OSINT, threat intelligence, and general security awareness.

⚠️ **Disclaimer:** All session content is educational and awareness-focused. No real target, victim, or client data is included in any session materials.

---

## About

This repository archives the structure and key material from live sessions, so each talk becomes a lasting reference instead of disappearing after the event. It also includes the full planning system behind each session — topic bank, checklists, FAQ prep, and promotion templates — so the process is repeatable and consistent every time.
