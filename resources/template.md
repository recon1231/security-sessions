# Session Promotion Templates

Ready-to-copy announcement text for promoting upcoming sessions across platforms.

---

## X — Announcement (Arabic)

```
جلسة جديدة قريبًا 🎙️

هتكلم عن: [موضوع الجلسة]

هنغطي:
• [نقطة 1]
• [نقطة 2]
• [نقطة 3]

الموعد: [التاريخ والوقت]
هتكون [أونلاين / حضوري] — التفاصيل هنا: [رابط]

مين حابب ينضم؟ 👇
```

## X — Announcement (English)

```
New session coming up 🎙️

Topic: [Session Topic]

We'll cover:
• [Point 1]
• [Point 2]
• [Point 3]

When: [Date/Time]
[Online / In-person] — details here: [link]
```

## LinkedIn — Announcement

```
I'll be running a session on [topic] on [date].

We'll cover [brief 1-2 sentence description of what the session includes and who it's for].

Details/registration: [link]

Would love to see some of my network there — feel free to share with anyone who might find it useful.
```

## Day-Of Reminder (X/Facebook)

```
Starting in [X hours/minutes] 🎙️

[Session topic] — join here: [link]
```

## Post-Session Recap

```
Thanks to everyone who joined today's session on [topic]!

Key takeaway: [one sentence summary]

Full session notes/writeup: [link, once published]

If you missed it, [link to recording/notes if available].
```

---

*Fill in brackets per session — keep the structure consistent so your audience recognizes the format.*
